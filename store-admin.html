¿Cómo funciona?
1. Guardá tu API key de ImgBB arriba.
2. Para cada producto, hacé clic en la foto → elegí una imagen desde tu computadora → se sube automáticamente a ImgBB.
3. También podés pegar directamente la URL de una imagen que ya tengas.
4. La imagen queda guardada y se muestra en la tienda al instante (sin recargar nada).
